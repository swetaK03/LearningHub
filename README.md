# Learning Hub Chat Application

Welcome to **Learning Hub Chat**, a modern web application offering real-time, dynamic chat experiences for users! Built with Django, Django Channels, and WebSockets, it combines powerful backend technologies with an intuitive interface for seamless interaction.

## 🌟 Highlights at a Glance

- **Real-time Chat:** Instantly connect with other users via WebSockets.
- **Secure Authentication:** Sign up and log in with ease.
- **Persistent History:** Access your past conversations anytime.
- **Interactive User Menu:** Easily find and chat with other registered users.
- **User-Friendly Design:** Enjoy a clean, collapsible sidebar and intuitive chat interface.

## 🚀 Technologies Behind the Magic

### Backend:
- **Django**: High-level web framework for rapid development.
- **Django Channels**: Enabling WebSockets for real-time interactions.
- **Django REST Framework**: API toolkit for data exchange.
- **SQLite**: Default database, adaptable to PostgreSQL if needed.

### Frontend:
- **HTML, CSS, JavaScript**: For responsive, interactive user experiences.
